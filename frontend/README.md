# Frontend

## Purpose
React dashboard for:
- live lamp monitoring
- RGB status display
- sensor data display
- manual override commands
- alert viewing

## Run locally
```bash
npm install
npm run dev
```

## Important files
- `src/main.jsx` - app entry point
- `src/App.jsx` - main layout
- `src/components/` - reusable UI
- `src/pages/` - dashboard pages
