# Scope and KPIs

## Scope
- Two ESP32 lamp nodes
- Ambient light sensing
- Motion sensing
- RGB lighting output
- MQTT communication
- Node.js backend
- React dashboard
- PostgreSQL storage
- OpenStack-oriented deployment

## KPIs
- Node-to-dashboard latency
- Command-to-actuation latency
- Correct state transitions
- Stable MQTT message flow
- Clear dashboard visibility of system state
